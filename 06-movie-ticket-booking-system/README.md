# Movie Ticket Booking System (CLI)

**Type:** CLI, state-based booking system

## Description

Command-line application to simulate movie ticket booking with seat availability management.

## Features

- Display available and booked seats
- Book a seat
- Prevent double booking
- Cancel booking
- Price calculation per seat

## Concepts Used

- Dictionaries
- State management
- Validation
- Functions
- Control flow

## How to Run

```bash
python main.py
```
