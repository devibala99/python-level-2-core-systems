# Banking System (CLI)

**Type:** CLI, Multi-user transaction system

## Description
Menu-driven Python application that simulates a basic banking system with multiple user accounts and secure transaction handling.

## Features
- Create bank accounts
- Deposit and withdraw money
- Balance inquiry
- Transaction history
- Input validation for amounts and accounts

## Concepts Used
- Dictionaries
- Functions
- Loops
- Validation
- State management

## How to Run
```bash
python main.py
