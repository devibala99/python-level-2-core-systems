# Voting System (CLI)

**Type:** CLI, rule-based application

## Description

Menu-driven Python application to conduct a simple voting process with strict rule enforcement.

## Features

- Register voters
- One-user-one-vote enforcement
- Candidate-wise vote counting
- Result display

## Concepts Used

- Dictionaries
- Sets
- Functions
- Conditional logic
- Validation

## How to Run

```bash
python main.py
```
