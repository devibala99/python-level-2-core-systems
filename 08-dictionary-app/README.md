# Word Suggestion System (CLI)

## Type

Python CLI Project

## Description

A simple command-line based word suggestion system that suggests words from a predefined dictionary based on user input.  
The system prioritizes words that start with the given fragment and also supports partial matches within words.

## Features

- Accepts partial word input from the user
- Suggests words based on prefix matching
- Falls back to substring matching if needed
- Case-insensitive input handling
- Clean separation of logic using utils.py

## Concepts Used

- Functions
- Lists
- String methods
- Conditional logic
- Looping
- Input validation
- Modular programming (main.py & utils.py)

## How to Run

1. Ensure Python 3 is installed
2. Clone or download the project
3. Run the program:
   ```bash
   python main.py
   ```
