# Expense Tracker (CLI)

## Type
Python CLI Application

## Description
A simple command-line based Expense Tracker that allows users to record daily expenses, view all expenses, and analyze spending through summaries.

## Features
- Add expense with amount, category, note, and date
- View all recorded expenses
- Calculate total expenses
- Show category-wise expense totals
- Identify the highest expense

## Concepts Used
- Lists and dictionaries
- Functions and modular design
- Input validation
- Date handling
- Aggregation and calculations
- Separation of logic (main.py & utils.py)

## How to Run
1. Ensure Python is installed
2. Keep `main.py` and `utils.py` in the same folder
3. Run the program using:

   ```bash
   python main.py
