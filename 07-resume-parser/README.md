# Resume Parser (CLI)

**Type:** CLI, text-processing application

## Description

A command-line based Resume Parser that reads a plain text resume file and extracts key information using string operations and regular expressions.

## Features

- Read resume from `.txt` file
- Extract name, email, and mobile number
- Identify skills using keyword matching
- Detect basic education details
- Handle missing fields safely

## Concepts Used

- File handling
- Regular expressions (regex)
- String parsing
- Functions and modular design
- Lists and dictionaries

## How to Run

```bash
python main.py
```
