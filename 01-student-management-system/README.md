# Student Management System (CLI)

**Type:** CLI, CRUD-based application

## Description

Menu-driven Python application to manage student records using in-memory data structures.

## Features

- Add / View / Update / Delete students
- Unique Student ID generation
- Input validation
- Modular structure

## Concepts Used

- Functions
- Dictionaries
- Loops
- Validation
- Error handling

## How to Run

```bash
python main.py
```
