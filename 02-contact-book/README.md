# Contact Book (CLI)

**Type:** CLI, CRUD-based application

## Description

Menu-driven Python application to store, search, update, and delete contact details using in-memory data structures.

## Features

- Add new contacts
- View contact by Contact ID
- Update existing contact details
- Delete contacts
- Unique Contact ID generation
- Input validation

## Concepts Used

- Functions
- Dictionaries
- Loops
- String handling
- Validation
- Error handling

## How to Run

```bash
python main.py
```
