# To-Do List Manager (CLI)

**Type:** CLI, State-based application

## Description

Menu-driven Python application to manage daily tasks with priority and status tracking.

## Features

- Add tasks
- View all tasks
- Update task status (pending / done)
- Delete tasks
- Priority-based task handling

## Concepts Used

- Lists and dictionaries
- Functions
- State transitions
- Input validation
- Control flow

## How to Run

```bash
python main.py
```
